# Quoll
Quoll is the bot that I made for the Mine-Crack discord server.


### Commands

- `!kick <player>` kicks a player

- `!ban <player>` bans a player

### Other functions

- randomly sends funny GIFs in the chat
- Deletes chat messages from #minecraft channel
- Deletes private whisper messages from #server-console channel
- Copies announcements to the #mnecraft channel
- Makes fun of Rylan for bragging
